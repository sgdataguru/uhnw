# User Story: 16 - View Alternate Investment Penetration

**As a** Managing Partner,
**I want** to see alternate investment penetration percentage across client segments,
**so that** I can identify opportunities to expand alternate product adoption.

## Acceptance Criteria

* Dashboard displays alternate investment penetration rates
* Breakdown by client segment (AUM tiers, regions, sectors)
* Comparison against benchmarks or targets
* Visual trend showing penetration changes over time
* Drill-down to see which alternates are most/least adopted
* Highlights under-penetrated segments with high potential

## Notes

* Page 1 highlights: "Alternate penetration % across client segments"
* Alternates include private equity, venture capital, real estate, etc.
* Key metric for diversification strategy
