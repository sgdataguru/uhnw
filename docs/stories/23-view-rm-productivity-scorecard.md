# User Story: 23 - View RM Productivity Scorecard

**As a** Managing Partner,
**I want** to view an RM productivity scorecard showing calls, conversions, and response times,
**so that** I can monitor team performance and identify coaching opportunities.

## Acceptance Criteria

* Scorecard displays key productivity metrics per RM
* Metrics include: calls made, meetings held, conversions, response times
* Comparison against team averages and targets
* Trend analysis showing performance over time
* Ranking or leaderboard view across RMs
* Drill-down to activity details for each RM
* Supports team of 30-35 RMs

## Notes

* Additional elements: "RM productivity scorecard (calls, conversions, response times)"
* Helps management identify top performers and those needing support
* Not meant to replace relationships but to improve visibility
