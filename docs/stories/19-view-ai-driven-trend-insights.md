# User Story: 19 - View AI-Driven Trend Insights

**As a** Relationship Manager,
**I want** to see AI-driven insights on trending patterns and key graphs,
**so that** I can understand market and client trends without manual analysis.

## Acceptance Criteria

* Dashboard displays key trending graphs with AI-generated insights
* Insights explain what the trend means and potential implications
* Automatically highlights anomalies or significant changes
* Trends cover: market movements, client behavior, sector performance
* Insights are refreshed as new data is ingested
* User can drill down into specific trends for more detail

## Notes

* Page 2 additions: "Key trending graphs with AI-driven insights"
* AI should surface actionable insights, not just raw data
* Reduces cognitive load for RMs by pre-analyzing trends
